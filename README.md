# 知产管家 · IP Butler

> 商标 + 软件著作权申请一站式工具。纯本地、零依赖、双击即用。

---

## 快速开始

### 安装

无需安装。整个项目是**纯静态 HTML**，双击 `index.html` 即可在浏览器中使用。

> 推荐使用 **Chrome / Edge** 浏览器（IndexedDB 在这两个浏览器里最稳定）。

### 第一次使用

1. 双击打开 `index.html`
2. 进入「申请人」 → 添加你的身份信息（个人 / 企业均可）
3. 进入「商标」或「软著」 → 新建项目
4. 编辑完点「保存」，再点「导出申请书 PDF」

## 功能清单

### ✅ 商标管理
- Nice 分类 45 类完整收录
- 类似群组 + 典型商品/服务参考
- 商标图样上传（图形 / 组合商标必须）
- 商品/服务项目勾选自动汇总
- **一键生成符合商标局格式的申请书 PDF**

### ✅ 软著管理
- 软件基本信息登记
- **源程序自动按规则切分**（前 N 页 + 后 N 页，每页 50 行）
- 一键生成 60 页源程序 PDF（含封面、自动分页）
- 一键生成符合版权中心格式的申请表 PDF
- 支持多文件 / 粘贴文本

### ✅ 申请人管理
- 个人 / 企业 双模式
- 支持「默认申请人」自动填入
- 一个申请人复用多个项目

### ✅ 文件库
- 源程序 / 说明书 / 商标图样集中管理
- 上传后自动关联到项目
- 一键下载

### ✅ 数据安全
- **所有数据存在浏览器本地**（IndexedDB）
- 不上传任何服务器
- 「导出全部数据」一键 JSON 备份
- 「导入数据」可恢复

## 商标申请全流程（简版）

1. **查重** — 去 [sbj.cnipa.gov.cn](https://sbj.cnipa.gov.cn) 检索有没有相同/近似商标
2. **录入本工具** — 申请人 / 商标信息 / 类别 + 商品 / 图样
3. **导出申请书 PDF** — 作为线下提交参考
4. **在线提交** — 注册账号、数字证书，提交到商标局网上系统
5. **缴费** — ¥270/类（10 项以内），超项 ¥30/项
6. **跟踪** — 在本工具更新状态（受理 / 审查 / 公告 / 注册）
7. **续展** — 注册有效期 10 年，到期前 12 个月开始续展

---

## 软著申请全流程（简版）

1. **在本工具录入** — 软件基本信息 + 著作权人
2. **上传源程序** — 自动生成 60 页源程序 PDF
3. **写说明书** — 6 页以上，另存 PDF 上传
4. **导出申请表 PDF** — 在版权中心系统填报前作为草稿参考
5. **在线提交** — [ccopyright.com](https://www.ccopyright.com) 注册账号 → 填表 → 上传
6. **缴费** — ¥100（个人）/ ¥250（企业）
7. **等待 30-60 工作日** — 登记证书邮寄

> ⚠️ 本工具生成的 PDF 是**申请材料草稿**，最终提交以商标局 / 版权中心在线系统为准。

## 隐私声明

- **营业执照图片**会上传到后端 AI 视觉服务进行识别（通过本站 HTTPS 代理）
- **GitHub 仓库信息**会通过 GitHub API 获取
- **API Key** 保存在服务端，浏览器不持有
- **其他数据**（项目、申请人、源码）保存在浏览器本地（IndexedDB）
- 后端不存储营业执照图片，仅做识别代理转发
- 关闭浏览器数据 / 切换电脑 = **数据丢失**（请用「导出全部数据」定期备份）
